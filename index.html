Help

クイック追加

  1. 登録したい URL をコピーする
  2. このアプリの画面上で ⌘/Ctrl V で貼り付ける（入力欄の外で OK）。上部にクイック追加バーが開き、URL とタイトル候補が自動で入る
  3. Enter を押すと登録される

タイトルは編集可。グループ・タグ・メモは →（入力末尾）か「詳細」でフォームへ。空のまま開くには ⌘/CtrlShiftL。

キーボードショートカット

⌘/CtrlV
URL を貼り付けてクイック追加バーを起動
⌘/CtrlShiftL
クイック追加バーを開く（貼り付け待ち）
/
検索ボックスにフォーカス
n
新規リンク追加モーダルを開く
?
このヘルプを開く
Esc
クイック追加バー / モーダル / メニューを閉じる
⌘/CtrlEnter
リンク / グループの追加・編集フォームで登録

グループ / タグの入力欄では ↑↓ で候補を移動、Enter で確定、Tab で次の項目へ移動できます。
